# Musistruct
Android Basics Nanodegree Project 4

### Goal of the app
This app was part of the Android Basics Nanodegree at <a href="http://udacity.com">udacity.com</a>. The goal of the app was to create the a list of music albums and a detail screen for every album with its information. The app is not functioning as a media player! The exercise was about ListViews and Intents to launch activities.

Used techniques:
- Custom class
- Custom Adapter
- ListView
- Explicit intent with extras
